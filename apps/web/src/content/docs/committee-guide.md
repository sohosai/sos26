# 実委人ガイド

実行委員として企画の管理やフォームの作成を行うための操作方法を説明します。

## マスターシート

全企画の情報を一覧で確認・編集できます。

- 「マスターシート」ページから全企画のデータを閲覧できます
- セルをクリックして直接編集が可能です
- CSVエクスポートにも対応しています

## フォームの作成

企画人に対してフォームを配信できます。

1. 「フォーム」ページから「フォームを作成」を選択します
2. フォームのタイトル・説明・質問項目を設定します
3. 提出期限を設定して公開します

## お知らせの配信

企画人に対してお知らせを送信できます。

1. 「お知らせ」ページから「お知らせを作成」を選択します
2. タイトル・本文を入力します
3. 配信対象を選択して送信します

## メンバー管理

実行委員のメンバー管理が行えます。

- 「メンバー管理」ページから実行委員の一覧を確認できます
- 役割の変更や権限の管理が可能です
