# はじめに

雙峰祭オンラインシステム（SOS）へようこそ。このシステムは、筑波大学の学園祭「雙峰祭」の企画運営を支援するためのWebアプリケーションです。

## アカウント登録

1. トップページの「新規登録」ボタンからアカウントを作成します
2. 筑波大学のメールアドレス（`@u.tsukuba.ac.jp`）を入力してください
3. 届いた確認メールのリンクをクリックして、メールアドレスを認証します
4. プロフィール情報を入力して登録完了です

## 企画への参加

アカウント登録が完了したら、企画に参加できます。

- **招待コードで参加**: 企画の代表者から共有された招待コードを入力して参加します
- **新規企画の作成**: 新しく企画を立ち上げる場合は、「企画を作成」から登録できます

## 主な機能

### メンバー管理

企画のメンバー一覧の確認や、招待コードの発行ができます。

### フォーム

実行委員会から届くフォームへの回答ができます。提出期限を確認して、期限内に回答してください。

### お知らせ

実行委員会からの重要なお知らせを確認できます。

## 困ったときは

- サイドバーの「不具合報告」からお問い合わせできます
- ログインできない場合は、パスワードリセットをお試しください
