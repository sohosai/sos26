# 企画人ガイド

企画の登録から当日の運営まで、企画人としての基本的な操作方法を説明します。

## 企画の作成

1. サイドバーの企画セレクターから「企画を作成」を選択します
2. 企画名・企画概要を入力して作成します
3. 作成後、招待コードが発行されます

## メンバーの招待

企画にメンバーを追加するには、招待コードを共有してください。

1. 「メンバー管理」ページを開きます
2. 招待コードをコピーして、メンバーに共有します
3. メンバーが招待コードを入力すると、企画に参加できます

## フォームへの回答

実行委員会から届くフォームに回答してください。

- フォーム一覧は「フォーム」ページで確認できます
- **提出期限**を過ぎると回答できなくなるため、早めの提出をお願いします
- 提出後も期限内であれば回答の修正が可能です

## お知らせの確認

実行委員会からの連絡は「お知らせ」ページに届きます。定期的に確認してください。
